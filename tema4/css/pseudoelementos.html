Pseudo-elementos

Hay otros selectores que aplican a partes específicas de un elemento HTML. Por ejemplo, la primera letra, la primera línea, antes o después del elemento, etc. Esas partes se llaman pseudo-elementos y se utilizan así: selector::pseudo-elemento { ... }.

<blockquote>Estamos dando estilo a una cita importante sobre CSS</blockquote>
blockquote {
  font-family: Georgia, serif;
  font-size: 19px;
  font-style: italic;
  background: #eee;
  margin: 19px;
  padding: 19px;
  border-radius: 19px;
}
blockquote::first-letter, blockquote::after {
  font-size: 29px;
  font-weight: 800;
}
blockquote::after {
  content: '❞';
}

⁉️ Ejercicio: Añade estas comillas ❝ al principio de la cita utilizando solo CSS.
